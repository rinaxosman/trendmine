# TrendMine

TrendMine turns public trend data from Reddit and Google Trends into practical business and side hustle ideas.

## How it works
- Fetches public Reddit posts from selected subreddits
- Fetches public Google Trends data usingRSS
- Analyzes trends and clusters them into themes
- Generates actionable business ideas backed by evidence (using the signals)

## Tech
- React + typescript
- Supabas
- Public data sources

## Running locally

```bash
npm install
```
```bash
npm run dev
```
